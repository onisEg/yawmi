# يومي 🌙

تطبيق شخصي لمتابعة اليوم: الصلوات، العادات، اليوميات، المزاج، المهام، والمصاريف — كله بالعربي وشغال أوفلاين.

## التشغيل لأول مرة

1. اعمل حساب على github.com لو مش عندك.
2. اعمل Repository جديد باسم `yawmi` وخليه Public.
3. ارفع كل الملفات دي فيه (Add file → Upload files).
4. من Settings → Pages → اختار Branch: main → Save.
5. استنى دقيقة، هيظهرلك لينك شكله: `https://اسمك.github.io/yawmi/`
6. افتح اللينك من موبايلك → من قائمة المتصفح اختار **"Add to Home Screen"**.

## التحديث

أي تعديل على `index.html` (من github.com مباشرة أو برفع نسخة جديدة) بيوصل للتطبيق تلقائي أول ما تفتحه وانت متوصل بالنت. مفيش تسطيب ولا حاجة.

## النسخ الاحتياطي (مهم!)

البيانات متخزنة على جهازك بس. كل أسبوع: ⚙ الإعدادات → **تصدير نسخة احتياطية** → احتفظ بالملف على Google Drive. لو غيرت موبايلك، استخدم **استيراد نسخة** وهترجع كل حاجة.
